# DeoVR — Landing Page Redesign

A modern, premium, conversion-focused redesign of the DeoVR download page, built entirely with AI-assisted development as part of an AI Design Challenge.

**Live demo:** https://deovr-landing-redesign.vercel.app

**Repository:** https://github.com/gregorytigunov-oss/deovr-landing-redesign

---

## Overview

The goal wasn't to redesign DeoVR from scratch, but to improve how the product is presented.

Instead of following the generic AI-generated SaaS aesthetic, I focused on creating a more immersive landing page centered around the player itself. The redesign emphasizes large-scale media, motion and editorial typography while preserving DeoVR's existing visual identity.

The redesign prioritizes showing the product over describing it. Whenever possible, the interface lets the player demonstrate its capabilities instead of relying on long feature lists.

---

## Goals

- Preserve the existing DeoVR brand language.
- Improve visual hierarchy and readability.
- Create a stronger first impression.
- Focus attention on the product itself.
- Increase perceived product quality through motion and interaction.
