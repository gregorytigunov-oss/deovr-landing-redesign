# DeoVR — Landing Page Redesign

A modern, conversion-focused redesign of the DeoVR landing page created as an AI-assisted UI/UX improvement challenge.

Live demo: https://deovr-landing-redesign-4zyyzuf7q-gregory8.vercel.app/

---

## What this is

This project is a redesigned landing page for DeoVR — a cross-platform VR video player.

The goal was to improve:
- Visual hierarchy
- Conversion flow
- Readability
- Overall “premium” feel
- UX clarity while keeping original brand identity

---

## Key Improvements

### Hero Section
- Clear value proposition
- Strong primary CTA
- More focused layout and hierarchy

### Layout & Structure
- Improved section spacing
- Better content grouping
- More readable information flow

### UX Enhancements
- Simplified decision path
- Stronger call-to-action emphasis
- More modern landing page structure
